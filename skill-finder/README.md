# Skill Finder

Find and evaluate Claude skills using semantic search and quality scoring.

## Install

```
/plugin marketplace add tenequm/claude-plugins
/plugin install skill-finder@tenequm-plugins
```

## Example Prompts

```
"Find skills for building REST APIs"
"Search for React testing skills with high quality scores"
"Evaluate this skill against Anthropic best practices"
"Compare skills for Kubernetes deployment"
```
