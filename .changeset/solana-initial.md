Initial release of unified Solana development toolkit

- **solana-development**: Build Solana programs with Anchor framework or native Rust
  - Complete Anchor framework guide with macros, IDL, and TypeScript clients
  - Native Rust development patterns with manual account handling
  - 18 detailed reference files covering accounts, PDAs, CPIs, tokens, testing, deployment

- **solana-security**: Systematic security auditing framework for Solana programs
  - 5-step security review process for both Anchor and native Rust
  - Comprehensive security checklists and vulnerability patterns
  - 7 detailed reference files
