# Chrome Extension WXT

Build Chrome extensions with WXT framework. TypeScript, React, Vue, or Svelte.

## Install

```
/plugin marketplace add tenequm/claude-plugins
/plugin install chrome-extension-wxt@tenequm-plugins
```
