# GitHub CLI

Remote repository analysis, file fetching, and code discovery without cloning.

## Install

```
/plugin marketplace add tenequm/claude-plugins
/plugin install gh-cli@tenequm-plugins
```

## Example Prompts

```
"Fetch the main config file from repo X without cloning"
"Compare implementations between two repositories"
"Find trending Go repos with >1000 stars"
"Search for authentication patterns across GitHub"
```

14 references: repos, PRs, issues, releases, actions, search, remote analysis, discovery, extensions.
